help How to setup Google Sheets integration
  1. Go to Google Cloud Console
  2. Create a new project or select existing one
  3. Enable Google Sheets API
  4. Create a Service Account and download JSON credentials
  5. Share your spreadsheet with the service account email
  6. Copy the spreadsheet ID from the URL

Sync leads to your Notion workspace

help How to setup Notion integration
  1. Go to Notion Integrations
  2. Click "New integration" and give it a name
  3. Copy the "Internal Integration Token"
  4. Create a new database in Notion
  5. Share the database with your integration
  6. Copy the database ID from the URL
